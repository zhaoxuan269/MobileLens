## 使用方法

### 方案 A：使用默认可视化（推荐）

**当前配置**：
- ARMarkerTracker 对象已配置 MarkerVisualizer 组件
- 参数已优化，无需额外配置

**测试步骤**：
1. Build 到 iPad
2. 点击按钮启动 Marker 检测
3. 对准 ArUco Marker
4. 应该立即看到大号黄色发光立方体

---

### 方案 B：切换到增强版（可选）

**如果想要更酷炫的效果**：

1. 在 Unity 中找到 ARMarkerTracker 对象
2. 禁用 MarkerVisualizer 组件（取消勾选）
3. Add Component → Enhanced Marker Visualizer
4. 参数保持默认即可（已优化）
5. 重新 Build

**效果对比**：
- 默认版：单个大号黄色立方体（简洁明了）
- 增强版：中心脉动 + 4 个轨道立方体（酷炫华丽）

## 故障排查

### 问题 1：还是看不到立方体

**检查项**：
1. Console 是否有"✓ 在 Marker 上生成可视化对象"日志？
   - 如果没有：MarkerVisualizer 组件可能未启用
   - 如果有：立方体已生成，可能被遮挡

2. 尝试不同角度和距离：
   - 建议距离：50cm
   - 从上方俯视 Marker
   - 确保 Marker 平放

3. 检查光线：
   - 环境太暗可能看不到（虽然有发光效果）
   - 建议在明亮环境测试

### 问题 2：立方体闪烁或消失

**原因**：Marker 追踪不稳定

**解决**：
- 确保 Marker 清晰可见
- 改善光线条件
- 保持相机稳定
- 使用更高质量的打印 Marker

### 问题 3：想要更明显的效果

**可以尝试**：
1. 在 Unity 中增大 Height Offset（比如改为 0.3）
2. 使用 EnhancedMarkerVisualizer
3. 自定义 Visual Prefab（拖入自己的 3D 模型）
